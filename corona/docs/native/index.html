Native (Enterprise) API Reference

Current Public Release (2013.2076)

This is the complete list of native APIs for Corona Enterprise. If you're looking for Lua APIs, go to SDK API Reference.

Overview

Corona Enterprise allows you to extend Corona SDK's Lua API with your own extensions. You can wrap native features and expose them in Lua.

To get started, check out the Corona Enterprise Guide.

Frameworks

The native API is a superset of the base Corona SDK's Lua API.

There are 2 kinds of frameworks. There's a core set of C APIs (left) and platform-specific APIs (right):
