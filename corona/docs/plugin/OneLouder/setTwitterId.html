Overview

Sets a Twitter ID of user.

Syntax

OneLouder.setTwitterId(TwitterId)
TwitterId (required)

TYPE. It should be the Twitter ID of the user of the app. Only pass this if you are collecting the email address for other reasons.

Examples

OneLouder.setTwitterId( "a@twitter.com" )
