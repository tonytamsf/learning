goprezzo.claimScore()

Type function
Library goprezzo.*
Return value Boolean
Revision Current Public Release (2013.2076)
Keywords
Sample code GoPrezzo Sample Code
See also http://www.goprezzo.com/corona

Overview

Should be called after checkIn() to claim the players score and load the goprezzo tournament views into the game and allow the player to enter tournaments.

Returns true if the score was claimed successfully and the view loaded and false if the score could not be claimed or view not loaded due to error or timeout.

Syntax

goprezzo.claimScore()

Examples

    local success = goprezzo.checkIn(1,1,gameScore)
    if (scoreSuccess == true) then
        goprezzo.claimScore()
    else
        -- show error, remove GO button
    end
