goprezzo.checkIn()

Type function
Library plugin.goprezzo.*
Return value Boolean
Revision Current Public Release (2013.2076)
Keywords
Sample code GoPrezzo Sample Code
See also http://www.goprezzo.com/corona

Overview

The checkIn function should be called when a player chooses to interact with the goprezzo tournament platform by clicking a GO button on a level over screen. This checks the score into tournament platform so the user can obtain go tournament points for their game play.

Returns true if the score was successfully checked in and false if the score could not be checked in due to error or timeout.

If an error occurs the user should be informed and the GO button disabled and hidden.

Syntax

goprezzo.checkIn( fragment, difficulty, gameScore )
fragment (required)

long. The fragment is a unique numeric identifier for an individual game level or phase of play that a score is submitted from.

difficulty (required)

int. The difficulty the game is being played at. If your game does not support different difficulty levels pass 0 as the difficulty parameter. Otherwise associate a unique integer value with each game difficulty eg. Easy - 0, Medium - 1, Hard - 2.

gameScore (required)

double. The gameScore is the players score for a level or phase of play that is to be submitted to the goprezzo tournament platform.

Examples


    local success = goprezzo.checkIn(1,1,gameScore)
