goprezzo.init()

Type function
Library plugin.goprezzo.*
Return value Boolean
Revision Current Public Release (2013.2076)
Keywords
Sample code GoPrezzo Sample Code
See also http://www.goprezzo.com/corona

Overview

The init function initialises the goprezzo plugin. It should be called when the game loads. The plugin requires an internet connection to create a secure session with the goprezzo tournament platform for the submission of encrypted game details and to integrate the goprezzo tournament views into the game for user interaction. If the initialisation succeeds the init returns true. If a connection to the platform cannot be established the init function returns false. The function returns false if either the networkTimeout or pageloadTimeout are exceeded.

Syntax

goprezzo.init( gameId, networkTimeout, pageloadTimeout, goprezzoListener )
gameId (required)

String. The unique goprezzo gameId for the game using the plugin.

networkTimeout (required)

Number. The number of milliseconds to wait for an internet connection to be established with the goprezzo tournament platform.

pageloadTimeout (required)

Number. The number of milliseconds to allow for loading the goprezzo tournament pages in the webview provided by the plugin.

goprezzoListener (optional)

Function.

Examples

    -- listener for onResumed event after claim score
    local goprezzoListener = function(event)
        if event == "finishedClaimScore" then
            print("We returned from claimScore.. Resume whatever the app was doing..")
        end
    end

    -- global variable for goprezzo session status
    hasGoPrezzoSession = false

    -- when initialising you can check sessionCreated first

    hasGoPrezzoSession = goprezzo.sessionCreated()

    if hasGoPrezzoSession == false then
        hasGoPrezzoSession = goprezzo.init( "cacf40ff7bd76411ca17e4f9b28788edc2fcf8f1d5f898e1a23d25a0c55de2da", 5000, 30000, goprezzoListener )
    end

    -- if the session has initialised
    if hasGoPrezzoSession == false then
        -- log no session
    else
        -- log session connected
    end
