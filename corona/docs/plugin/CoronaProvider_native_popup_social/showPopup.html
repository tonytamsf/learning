native.showPopup() > (Social Plugin)

Type function
Library PLUGIN_NAME.*
Return value [TYPE][api.type.TYPE]
Revision Current Public Release (2013.2076)
Keywords show, popup, social
Sample code
See also

Overview

Displays the operating system's default popup window for a specified service.

Gotchas

The Android version of this plugin does not support Facebook. This is due to a bug at Facebook's end. For more information see the following link: https://developers.facebook.com/bugs/332619626816423/

Android support is currently in Beta phase.

Syntax

native.showPopup( name, options )
name (required)

String. The string name of the popup to be shown. It must be set to the following: "social"

options (optional)

Table. Table of parameters. See Format for options below.

Format for options

The options table contains the following properties:

service (required)

String. (Applies to iOS only) The name of the social service popup you wish to show. Supported values are: "twitter", "facebook", "sinaWeibo" This key is ignored if specified on Android.

image (optional)

Table. This is a table of the form { baseDir=, filename= } to the image you wish to post.

message (optional)

String. This is a string that prepopulates the message field of the popup.

listener (optional)

Listener. This is a listener that supports popup events.

url (optional)

String. This is a string for the url you wish to post. Alternatively, this can be an array of url strings.

Examples

local popupName = "social"
local serviceName = "twitter" -- Supported values: "twitter", "facebook", "sinaWeibo"

local isAvailable = native.canShowPopup( popupName, serviceName )

if isAvailable then
    local listener = {}
    function listener:popup( event )
        print( "name(" .. event.name .. ") type(" .. event.type .. ") action(" .. tostring(event.action) .. ") limitReached(" .. tostring(event.limitReached) .. ")" )
    end

    native.showPopup(
        popupName,
        {
            service = serviceName,
            message = "hi there!",
            listener = listener,
            image = 
            {
                { filename = "world.jpg", baseDir = system.ResourceDirectory },
                { filename = "bkg_wood.png", baseDir = system.ResourceDirectory },
            },
            url = 
            { 
                "http://www.apple.com",
                "http://www.coronalabs.com",
                "http://docs.coronalabs.com",
                "http://developer.coronalabs.com" 
            }
        } )
else
    native.showAlert(
        "Cannot send " .. serviceName .. " message.",
        "Please setup your " .. serviceName .. " account or check your network connection",
        { "OK" } )
end
