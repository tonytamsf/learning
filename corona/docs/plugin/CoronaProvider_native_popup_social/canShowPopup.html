native.canShowPopup() > (Social Plugin)

Type Library Return value Revision Keywords Sample code See also Function native.* Boolean Current Public Release (2013.2076) show, popup, social

Overview

Returns whether or not the popup type can be shown.

iOS Behavior By "can be shown", we usually mean whether the service in question is setup on the device. This means that the user has set up for example a Twitter account via the Settings app on iOS

Android Behavior By "can be shown", we usually mean whether the service in question is installed on the device.

Syntax

native.canShowPopup( name, service )
name (required)

String. The string name of the popup to be shown. It can be one of the following: "social"

service (required)

String. iOS: The name of the social service popup you wish to show. Supported values are: "twitter", "facebook", "sinaWeibo" Android: The package name of the social service you wish to check is installed on the device. For example, if you pass "twitter" as the service, the canShowPopup function will check to see if any package names installed on the device contain the string "twitter" and if a match is found, it will return true.
