tapfortap.prepareInterstitial()

Type function
Library tapfortap.*
Return value None
Revision Current Public Release (2013.2076)
Keywords ad, fullscreen, tapfortap
Sample code
See also tapfortap.showInterstitial(), tapfortap.setInterstitialListener(function), tapfortap.interstitialIsReady()

Overview

Prepares an interstitial ad. This can optionally be called before the first call to tapfortap.showInterstitial(). Once tapfortap.showInterstitial() has been called, it will automatically prepare the next interstitial.

Syntax

tapfortap.prepareInterstitial()

Examples

local tft = require "tapfortap"
tapfortap.initialize("My Tap for Tap API key")

-- Prepare an interstitial for the first call to showInterstitial()
tapfortap.prepareInterstitial()

...

-- Later show an App Wall
tapfortap.showInterstitial()

...


-- At some point even later show another interstitial. We do not need to call prepareInterstitial() again as the previous call the showInterstitial() prepared the next interstitial automatically.
tapfortap.showInterstitial()
