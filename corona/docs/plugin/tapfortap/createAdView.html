tapfortap.createAdView(verticalAlignment, horizontalAlignment, xOffset, yOffset, scale)

Type function
Library tapfortap.*
Return value NONE
Revision Current Public Release (2013.2076)
Keywords ad, banner, tapfortap
Sample code
See also tapfortap.removeAdView, tapfortap.setAdViewListener(function)

Overview

Begins showing ads at the given location.

Syntax

tapfortap.createAdView(verticalAlignment, horizontalAlignment, xOffset, yOffset, scale)
verticalAlignment (required)

number. The vertical alignment of the ad. See "Vertical Alignment Values" below

horizontalAlignment (required)

number. The horizontal alignment of the ad. See "Horizonal Alignment Values" below

xOffset (optional)

number. The x offset of the ad from the givien horizontal position.

yOffset (optional)

number. The y offset of the ad from the given vertical position.

scale (optional)

number. How much to scale the ad. Ads are by default 320x50.

Vertical Alignment Values

  • Top: 1
  • Center: 2
  • Bottom: 3

Horizontal Alignment Values

  • Left: 1
  • Center: 2
  • Right: 3

Examples

local tft = require "tapfortap"

local TOP = 1
local CENTER = 2
local BOTTOM = 3
local LEFT = 1
local RIGHT = 3

tapfortap.initialize("My Tap for Tap API key")

-- Show ad at the bottom center of the display with no offsets or scale
tapfortap.createAdView(BOTTOM, CENTER)

-- Show ad at the bottom center of the display with no x and y offset and a scale of 1.5
tapfortap.createAdView(BOTTOM, CENTER, 0, 0, 1.5)

-- Show ad at the bottom center of the display with an x-offset of 20, a y-offset of 0 and a scale of 1.5
tapfortap.createAdView(BOTTOM, CENTER, 20, 0, 1.5)
