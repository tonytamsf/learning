tapfortap.setInterstitialListener(function)

Type function
Library tapfortap.*
Return value None
Revision Current Public Release (2013.2076)
Keywords event, ad, fullscreen, tapfortap
Sample code
See also tapfortap.prepareInterstitial(), tapfortap.showInterstitial(), tapfortap.interstitialIsReady()

Overview

Sets the interstitial's listener. This function will be called the when the interstitial fires and event.

Syntax

local function interstitialListener(event)
    ...
end

tapfortap.setInterstitialListener(interstitialListener \)
Listener Function (required)

function. The function that will handle the event fired by the interstitial. See "Interstitial Listener Function" below for more details.

Interstitial Listener Function

The listener function must accept a table as a argument. This table has the following keys:

  • name: The name of the libray passing the event. This will always be tapfortap.
  • event: The name of the event. Possible events include.
    • onReceiveAd: Fired when a new ad was received.
    • onShow: Fired when the ad was shown.
    • onDismiss: Fired when the ad was dismissed.
    • onFailToReceiveAd: Fired when the ad failed to download. This event message key with the failure reason.
  • message: Any extra data that the event passes along. Only the onFailToReceiveAd event uses the message key to send along the failure reason.

Examples

-- Create a listener function
local function interstitialListener(event)
    print(event.name, "interstitialListener " .. event.event .. " " .. event.message)
end

local tft = require "tapfortap"
tapfortap.initialize("My Tap for Tap API key")

-- Set the interstitial listener to the fuction above
tapfortap.setInterstitialListener(interstitialListener)
tapfortap.showInterstitial()
