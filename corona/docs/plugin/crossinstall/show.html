ads.show()

Type library
Library crossinstall.*
Revision Current Public Release (2013.2076)
Keywords CrossInstall
Sample code
See also
Availability Starter, Pro, Enterprise

Overview

ads.show() will display a CrossInstall ad at the given screen location as either a banner ad or a fullscreen interstitial, depending on the values specified.

Syntax

ads.show( adUnitType , placement, params )
adUnitType (required)

String. Either "banner" or "interstitial". An "interstitial" ad will display on the full screen along with an "X" button for the user to close the ad. A "banner" ad will display the ad you requested at the position you provide.

placementName (required)

String. You may provide a custom placement string that defines where you are placing the ad in your app. This will show up in your revenue reporting and is also used in the optimization engine.

params (required)

Table. Specify ad placement and size parameters. See Parameter Table Properties below.

Parameter Table Properties

The params table contains the following properties:

x ~^(required for "banner" option)^~

Number. x position where you would like the banner ad placed.

y ~^(required for "banner" option)^~

Number. y position where you would like the banner ad placed.

w ~^(required for "banner" option)^~

Number. Width of the banner ad you would like to display. Currently supported dimensions are: 180x300, 320x480, 300x300, 640x960, 300x250, 300x50, 320x50, 480x320, 960x640.

h ~^(required for "banner" option)^~

Number. Height of the banner ad you would like to display. Currently supported dimensions are: 180x300, 320x480, 300x300, 640x960, 300x250, 300x50, 320x50, 480x320, 960x640.

timeout (optional)

Number. An optional timeout value, in seconds, at which the CrossInstall ad will give up on trying to pull in an ad to display to your user. Defaults to 10 seconds.

Examples

local ads = require "ads"

local function adListener( event )
  -- event table includes:
  --    event.provider
  --    event.isError (e.g. true/false )
  --    event.response (e.g. "Success"/<errorMessage>/"Fullscreen closed")

  if event.isError then
    -- handle failed ad loading
  else
    -- optionally handle interstitial closing, etc
  end
end

ads.init( "crossinstall", "your crossinstall app id, like AbCd", adListener )

-- to display a 320x50 banner ad at the bottom of the screen:
ads.show( "banner", "my placement name", { x=0, y=430, w=320, h=50, timeout=15 } )

-- to display a fullscreen interstital ad:
ads.show( "fullscreen", "my placement name", {timeout=15} )
