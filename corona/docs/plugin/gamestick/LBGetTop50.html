Overview

gamestick.LBGetTop50() returns the top 50 players from this game's leaderboard.

Note: If less than 50 players are found, then the number of entries returned will be the number of entries found.

Syntax

gamestick.LBGetTop50( {requestID = 1}, listener)
options (required)

Table. A table of parameters.

listener (required)

TYPE. The listener function which will handle the API response.

Format for options

The options table contains the following properties:

requestID (required)

TYPE. An integer which will be returned with the response data as a property called "id". This is used to simplify matching up requests and responses when multiple requests are made close together

Examples

local function gamestickListener( event )

    --Failed to get a valid response from server
    if event.isError then
        native.showAlert( "Error: "..event.name, event.message, { "OK" } )
    else
        if event.name == "LBGetTop50" then
            local id  = event.id 

            if event.data then
                local data = event.data -- a table containing up to 50 entries
                for i = 1, #data do
                    local AvatarID = data[i].avatarID --number: user's unique ID number
                    local Name     = data[i].name --string: user's name
                    local Score    = data[i].score --number: user's current scores
                    local Position = data[i].position --number: user's position on the leaderboard
                end
            end
        end
    end
end


gamestick.LBGetTop50({requestID = 4}, gamestickListener)
