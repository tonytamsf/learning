apprever.requestBannerAds()

Type function
Library apprever.*
Revision Current Public Release (2013.2076)
Keywords AppRever, ads, banner ads
Sample code
See also

Overview

Serve an AppRever banner ad.

You can set the x, y location, height and width, animate the ad, and include an optional message.

Syntax

apprever.requestBannerAds( options )
options (required)

Table. Short description goes here. See Format for options below.

Format for options

The options table contains the following properties:

x (required)

Number. x location of top left of banner ad

y (required)

Number. y location of top left banner ad

w (optional)

Number. width of banner ad

h (optional)

Number. height of banner ad

m (optional)

String. Message associated with banner ad (stored in the AppRever database)

a (optional)

Number. a=1 for animaited ads, a=0 to turn off banner ad animation

Examples

``````lua

-- You can set the width and height of banner ads to any size you like. To keep them 
-- proportionate to the original 960x100 size, use the following code for landscape mode.
-- They are automatically scaled down proprtionally for portrait mode on each ad request. 
-- So for a perfect fit on any device:

-- Calculate your actual device width
local width = math.round((display.contentWidth-2*display.screenOriginX) / display.contentScaleX)

-- Now scale proportionally to the 960x100 ad units currently used by AppRever
local height = (width/960)*100     

apprever.requestBannerAds( {x=0, y = 0, w=width, h=height, m="beat level 10", a=1} )    
For banner ads at the bottom of your landscape-mode App, just add in a calculation to 
determine your actual device height, and for the y location of the ad, just subtract 
off the height of the ad itself:

local width = math.round((display.contentWidth-2*display.screenOriginX) / display.contentScaleX)

-- this scales proportionally to the 960x100 ad units currently used by AppRever
local height = (width/960)*100     

local actualScreenHeight = math.round((display.contentHeight-2*display.screenOriginY) / display.contentScaleY)

apprever.requestBannerAds( {x=0, y = actualScreenHeight-height,  w=width, h=height, m="beat level 10", a=1} )

``````
