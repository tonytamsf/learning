Overview

The plugin_demoLibrary plugin will show you IOS UIView with green background and a UILabel on it.Provided by ideskLearingTracker team. It'll help developers to send application data to us by using IOS PasteBoard.

Project Settings

SDK

When you build using the Corona Simulator, the server automatically takes care of integrating the plugin into your project.

All you need to do is add an entry into a plugins table of your build.settings. The following is an example of a minimal build.settings file:

settings =
{
    plugins =
    {
        -- key is the name passed to Lua's 'require()'
        ["plugin.demoLibrary"] =
        {
            -- required
            publisherId = "com.idesklearningtracker",
        },
    },      
}
