Overview

Syntax

inmobi.createInterstitialAd( options )

local bannerAd = Inmobi.createInterstitialAd()

  • load

    interstitialAd.load() Load method will load the interstitial ad.

  • show

    interstitialAd.show() Show method will display interstitial ad.
