Overview

Syntax

local analytics = inmobi.getAnalytics()

  • startSessionManually Evoke startSession when the application is launched. Do this before any other event is recorded, preferably in the onCreate() or onResume lifecycle methods of the launcher Activity.

    analytics.startSessionManually()

  • stopSessionManually Evoke endSession when the app is closed, typically in onDestroy() or onPause().

    analytics.stopSessionManually()

  • beginSection Evoke this event when the user has started viewing/playing a new Section. This could be a tab or app content in case of non-gaming apps. This could be a level in case of gaming apps.

    analytics.beginSection("sectionName" [, params])

  • endSection Evoke this event when a user has finished viewing/playing a section.

    analytics.endSection("sectionName" [, params])

  • tagEvent Use tagEvent to record any arbitrary event that is not covered by the above APIs.

    analytics.tagEvent("eventName" [, params])

Examples

-- 
-- InMobi Analytics
-- 
local analytics = Inmobi.getAnalytics();

-- startSessionManually
analytics.startSessionManually();

-- endSessionManually
analytics.endSessionManually();

-- beginSection
analytics.beginSection("startedlevel3", {userid="123"})

-- endSection
analytics.endSection("finishedlevel2", {userid="123"})

-- tagEvent
analytics.tagEvent("boughtGems", {userid="123"})
