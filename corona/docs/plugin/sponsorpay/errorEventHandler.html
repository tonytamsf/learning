sponsorpay.errorEventHandler()

Type function
Library sponsorpay.*
Revision Current Public Release (2013.2076)
Keywords
Sample code
See also

Overview

Used to register an error handler that is called when an error happen on the native SDK.

Syntax

sponsorpay.errorEventHandler( errorListener )
errorListener (required)

Listener. The listener that will be notified.

Format for event

The event table contains the following properties:

name (required)

String. The event name.

error (optional)

Table. The table containing the error information.

Format for error

The error table contains the following properties:

message (required)

String. The error message.

code (optional)

String. The error code (when applicable).

type (optional)

String. The error type (when applicable).

Examples

local function onSDKError( event )    
  local message = event.error.message 
  print( event.name .. "\n" .. message)
end

-- register the error event listener                                     
sponsorpay.errorEventHandler( onSDKError )
