System and OS Functions

This guide discusses the Corona system and os functions. Collectively, these are used to return system information and control system functions — get device information, enable multitouch, control the idle time, accelerometer, GPS, etc. — and deal with system time, date, and other OS-related information.

  • System Library
  • OS Library

System Library

Function   Description
system.getInfo() reference Returns information about the system on which the application is running.
system.openURL() reference Opens a web page in the device browser, creates an email, or calls a phone number.
system.pathForFile() reference Generates an absolute path using system-defined directories as the base. See Reading and Writing Files for more information.
system.getPreference() reference Returns a system preference value (ui or locale) as a string.
system.getTimer() reference Returns the time in milliseconds since application launch.
system.orientation reference Returns a string identifying the orientation of the device.
system.setTapDelay() reference Sets the delay time between when a tap is detected and when the tap event is delivered.
system.activate() reference Activates a system-level feature such as multitouch.
system.deactivate() reference Deactivates a system-level feature activated by system.activate().
system.scheduleNotification() reference Schedules a local notification event to be delivered at a future time.
system.cancelNotification() reference Removes the specified notification from the scheduler, status bar, or notification center.
system.setIdleTimer() reference Controls whether the system idle timer is enabled (automatic screen dim and sleep).
system.getIdleTimer() reference Determines whether the system idle timer is enabled.
system.setLocationAccuracy() reference Sets the desired accuracy of location (GPS) events to distance in meters.
system.setLocationThreshold() reference Sets the distance in meters between each location (GPS) event.
system.setAccelerometerInterval() reference Sets the frequency of accelerometer events.
system.setGyroscopeInterval() reference Sets the frequency of gyroscope events in Hertz.
system.hasEventSource() reference Returns whether the system delivers events corresponding to eventName.
system.vibrate() reference Vibrates the phone. On the Corona Simulator, this triggers a system beep.

OS Library

Function   Description
os.clock() reference Returns an approximation of the amount in seconds of CPU time used by the program.
os.date() reference Returns a formatted string or a table containing date and time.
os.time() reference Returns the current time in seconds from 1970, or the date and time specified by a table.
os.difftime() reference Returns the number of seconds from time "t1" to time "t2".
os.execute() reference Passes a string to the OS for execution and returns a system-dependent status code.
os.exit() reference Calls the C function exit() to terminate the host program.
os.remove() reference Deletes a file or directory in the documents or temporary directory.
os.rename() reference Renames a file or directory in the documents or temporary directory.
