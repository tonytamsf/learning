System Requirements

Mac OS X

  • Mac OS X 10.7 or later; 10.8 or later recommended (see notes below)
  • Intel-based Mac
  • Apple's iOS SDK and Xcode to create iOS device builds
Notes
  • Corona SDK on Mac supports building for both iOS and Android.
  • We recommend using the latest version of Mac OS X and Xcode to avoid device build issues. Currently there is a version of Xcode for OS X 10.7.x, but Apple might drop support for this in the near future.
  • You don't need to join the Apple Developer Program just to test your app in the Corona Simulator. You only need to install Apple's iOS SDK and Xcode and then build for the Xcode Simulator. You will, however, need to join the Apple Developer Program to test on actual iOS devices.
  • The Android SDK is not required to create Android device builds.
  • We only support Android devices that run Android 2.2 or higher with an ARMv7 processor. ARMv6 is not supported.

Ready to start?

Proceed to Installing Corona SDK — Mac OS X.

Windows

  • Windows 8, Windows 7, Vista, or XP
  • 1 GHz processor (recommended)
  • 76 MB of disk space
  • 1 GB of RAM (recommended)
  • OpenGL 1.3 or higher (available in most modern Windows systems)
Notes
  • Corona SDK on Windows only supports building for Android.
  • The Android SDK is not required to create Android device builds on Windows. You will, however, need to install the x86 (32-bit) version of the Java 6 Development Kit. Corona does not support JDK7.
  • We only support Android devices that run Android 2.2 or higher with an ARMv7 processor. ARMv6 is not supported.

Ready to start?

Proceed to Installing Corona SDK — Windows.
