Shapes: Paths, Fills, and Strokes

Shape objects are display objects whose geometry encloses an area defined by a boundary.

Creating Shapes

Corona makes it easy to create basic shapes with the following factory methods:

  • Rectangles: display.newRect()
    • Images: display.newImage()
    • Image Rects: display.newImageRect()
  • Rounded rectangles: display.newRoundedRect()
  • Circles: display.newCircle()

In addition, you can create arbitrary polygons shapes that are either convex or concave:

  • Polygons: display.newPolygon()

Paths

All shapes have a object.path property. This property lets you manipulate aspects of the shape.

  • CirclePath
  • RectPath
  • RoundedRectPath
  • [PolygonPath][api.type.PolygonPath]

Fill and Stroke

All shapes have a fill and a stroke. The fill of a shape is the interior area of the geometry, while the stroke of a shape is its boundary.

Specifying Paint

Corona uses the concept of paint to specify how fills and strokes are drawn.

Paints can be defined by Lua tables:

local redPaint = { 1, 0, 0 }
local bluePaint = { 0, 0, 1 }
local imagePaint = { type="image", filename="aquariumbackgroundIPhone.jpg" }
local sheetPaint = { type="image", sheet=mySheet, frame=3 }
local gradientPaint = { type="gradient", color1=redPaint, color2=bluePaint }

And can then be assigned to the shape's fill and stroke properties:

local circle = display.newCircle( halfW-w/2, halfH-h/2, 100 )
circle.x = display.contentCenterX
circle.y = display.contentCenterY

-- Fill with image
local imagePaint = { type="image", filename="aquariumbackgroundIPhone.jpg" }
circle.fill = imagePaint

-- Anti-alias stroke
local brush = { type="image", filename="brush.png"}
circle.stroke = brush

circle:setStrokeColor( 0, 1, 1 ) -- tint brush image
circle.strokeWidth = 30
