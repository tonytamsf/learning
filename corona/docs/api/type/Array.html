Overview

Lua arrays are just Lua tables except they contain only numerical indices and the indices are consecutive starting at 1. When inserting or removing values from Lua arrays, you should use the table.insert() and table.remove() library functions.

Note: There is no special 'Array' type in Lua. Instead, Lua tables can be either associative arrays (key-value pairs) and numerically-indexed arrays (consecutive values).

You should take care not to try to use a table as both kinds of arrays simultaneously. This can result in undefined behavior.

Table library

One confusing thing in Lua is that the library called table is a collection of functions used to manipulate Lua tables as (numerically-indexed) arrays.

Array vs Table

In Lua, there is no explicit 'Array' type. However, in this documentation we explicitly list it out.

Why? Mainly for clarity.

Some function parameters or results in the Corona API expect a numerically-indexed array. In those cases, we explicitly mention the type as 'Array' instead of 'Table' to emphasize that the table should be a pure numerical array.
