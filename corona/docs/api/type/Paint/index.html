Overview

Paint objects control the way in which geometry is rasterized.

Types of Paints

In general, paints are specified via a Lua tables containing the data.

There are various types of paints that can be specified:

  • Paint: solid colors
  • BitmapPaint: textures/images
  • CompositePaint: multitextures
  • GradientPaint: linear gradients
  • ImageSheetPaint: texture atlases

Specifying solid colors

Solid colors are specified by an array of color channels:

{ gray }
{ gray, alpha }
{ red, green, blue }
{ red, green, blue, alpha }

Properties

Color Tint

paint.r

Number. The red channel.

paint.g

Number. The green channel.

paint.b

Number. The blue channel.

paint.a

Number. The alpha channel.

Blends and Effects

paint.blendEquation

String. The blend equation per the OpenGL-ES 2 specification. This can be "add" (default), "subtract", or "reverseSubtract".

paint.blendMode

String. The blend mode. See shape.blendMode for a complete list of blend modes.

paint.effect

String. The name of the effect. See [guide.graphics.effects][Effects Guide] for a list of effects.
