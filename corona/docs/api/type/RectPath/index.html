Overview

Paths specify the geometry of shapes. Path objects give you access to manipulate this geometry.

Properties

(Inherits properties from Path)

path.height

Number. Height of the rectangle.

path.width

Number. Width of the rectangle.

Quadrilateral Distortion

The following properties allow you to distort the corners of a rectangle by specifying offsets.

The corners are ordered starting with the top-left and going around in counter-clockwise order (see diagram below).

You can learn more about 2.5D/3D effects in 2.5D/3D: Perspective, depth, and all that..

path.x1, path.y1

Number. The offset for the top-left corner. By default, (0,0).

path.x2, path.y2

Number. The offset for the bottom-left corner. By default, (0,0).

path.x3, path.y3

Number. The offset for the bottom-right corner. By default, (0,0).

path.x4, path.y4

Number. The offset for the top-right corner. By default, (0,0).
