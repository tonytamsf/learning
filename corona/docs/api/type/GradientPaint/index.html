Overview

A linear gradient is a linear progression from one color to another.

Specifying linear gradients

Linear gradients are specified by Lua tables:

local paint = {
    type = "gradient",
    color1 = {r,g,b,a},
    color2 = {r,g,b,a},
    direction = dir
}

local rect = display.newRect( 0, 0, 100, 100 )
rect.fill = paint
type (required)

String. This must be "gradient".

color1 (required)

Array. An array of color channels for the start color.

color1 (required)

Array. An array of color channels for the end color.

dir (optional)

String. The direction of the gradient from color1 to color2. This can be:

  • "down" (default)
  • "up"
  • "left"
  • "right"
