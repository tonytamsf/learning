object:setTextColor()

Type Function
Object TextObject
Library display.*
Return value none
Revision Current Public Release (2013.2076)
Keywords setTextColor, text, color
See also display.newText()
TextBox:setTextColor()
TextField:setTextColor()

Overview

Sets the color or gradient fill of a text object.

Syntax

Any of the following are valid:

object:setTextColor( gray )
object:setTextColor( gray, alpha )
object:setTextColor( red, green, blue )
object:setTextColor( red, green, blue, alpha )
object:setTextColor( gradient )
gray, red, green, blue, alpha (optional)

Numbers. Number between 0 and 255 that represent the corresponding value for that channel. alpha represents the opacity of the text object.

gradient (optional)

Object. Gradient as returned from the graphics.newGradient() function.

Examples

Solid Fill
local myText = display.newText( "Hello World!", 0, 0, native.systemFont, 32 )
myText:setTextColor( 255, 255, 255 )
Gradient Fill
local gradient = graphics.newGradient( { 255, 255, 255 }, { 200, 200, 200 }, "down" )

local myText = display.newText( "Hello World!", 0, 0, native.systemFont, 32 )

-- Apply gradient to text
myText:setTextColor( gradient )
