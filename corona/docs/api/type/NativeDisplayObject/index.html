Overview

NativeDisplayObjects are separate from Corona's OpenGL drawing canvas, and always appear above normal DisplayObjects. They are created with the following functions:

  • native.newMapView() (See also: Map)
  • native.newTextBox() (See also: TextBox)
  • native.newTextField() (See also: TextField)
  • native.newVideo() (See also: Video)
  • native.newWebView() (See also: WebView)
  • native.showWebPopup()
