TextField

Parent NativeDisplayObject
Library native.*
Revision Current Public Release (2013.2076)
Keywords text input, text field, native text
See also native.newTextField()

Overview

A native display object representing a single-line textfield for text input. For an example of how to create and use native TextField objects, see: native.newTextField()

NOTE: Native input text fields are only available in device builds and OS X Simulator, but not on the Windows Simulator.
