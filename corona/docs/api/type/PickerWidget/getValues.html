object:getValues()

Type Function
Object PickerWidget
Library widget.*
Return value Table
Revision Current Public Release (2013.2076)
Keywords widget, picker, pickerWheel, wheel, getValues
Sample code /CoronaSDK/SampleCode/Interface/WidgetDemo/
See also widget.newPickerWheel()

Overview

Returns a table with the value/index of the rows that are currently in the Picker Wheel selection area.

Syntax

object:getValues()

Example

The following example is for a Picker Wheel with 3 columns:

local selectedRows = myPicker:getValues()

for i = 1, #selectedRows do
    print( "Row", selectedRows[i].index, "value is:", selectedRows[i].value )
end
