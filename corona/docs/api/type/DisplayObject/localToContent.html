object:localToContent()

Type Function
Object DisplayObject
Library display.*
Return value Numbers (2)
Revision Current Public Release (2013.2076)
Keywords localToContent
Sample code /CoronaSDK/SampleCode/
See also

Overview

Maps given x and y coordinates (center) in local coordinates to content (e.g. stage) coordinates.

This method is useful for mapping the coordinates of a display object within a group to the content (viewable display screen).

This function returns two numbers: X and Y Content coordinates. Content coordinates are relative to the Top Left position (0,0) on the screen in both portrait and landscape modes.

NOTE: The values returned are relative to the Display Object's center point. The reference point is not used for calculating the returned values -- they are fixed at the object's center. (display.setReferencePoint will not change the returned value.)

Gotchas

This method is useful for comparing the location of display objects in different groups. When a display object is inserted into a group, the x,y properties for the object is relative to it's group (parent) and not the content (stage). object. object:localToContent() could be used to return the Content coordinates of each of the objects (from different groups) to determine the physical relationship between the objects.

This method is similar to object.conte=ntBounds, but only returns the x,y point of the object. (contentBounds returns the content coordinates of xMin, yMin, xMax, yMax).

Syntax

object:localToContent( x, y )
x, y (required)

Numbers. Coordinates relative to the display object's center point. Point [0,0] references the center point of the object.

Example

local group = display.newGroup() -- Create a group

local rect = display.newRect( 0, 0, 10, 20 ) -- Create a rectangle object
group:insert( rect ) -- add rect to group

group.x = 100; group.y = 200    -- move the group

rect:setReferencePoint( display.TopLeftReferencePoint )

rect.x = 5
rect.y = 10

rectContentX, rectContentY = rect:localToContent( 0, 0 )
groupContentX, groupContentY = group:localToContent( 0, 0 )

print( "group's position on screen: ", groupContentX, groupContentY )  -- prints 100, 200
print( "group.x,y: ", group.x, group.y )  -- prints 100,200

print( "rect's position on screen: ", rectContentX, rectContentY  )  -- prints 110, 220
print( "rect.x,y: ", rect.x, rect.y ) -- prints 5, 10
