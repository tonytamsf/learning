object.yScale

Type Number
Object DisplayObject
Library display.*
Revision Current Public Release (2013.2076)
Keywords y, scale, object scaling, size
See also object.xScale

Overview

Retrieve or change the scale of the object in the y-direction.

The scaling occurs around the object's reference point. The default reference point for most Display Objects is center.

You can flip a DisplayObject vertically by passing a negative value to object.yScale as seen in the example section below.

Gotchas

Value examples: 1.0 is 100%. 0.5 is 50%.

The xScale and yScale values can also be set by object:scale() function.

When a Display Object is converted into a physics object, scaling the Display Object will change the physical size but NOT affect the physics view of the object. This could result in unpredictable results when interacting with other physics bodies.

Example

local rect1 = display.newRect(25, 75, 50, 50)
rect1:setFillColor(170, 170, 170) 

-- Flip the rectangle vertically
rect1.yScale = -1

local rect2 = display.newRect(125, 75, 50, 50)
rect2:setFillColor(255, 255, 255)

rect2.yScale = 2  -- change the Y scale 200%
