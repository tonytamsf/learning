Library physics.*
Revision Current Public Release (2013.2076)
Keywords physics, bodies, body
Sample code /CoronaSDK/SampleCode/Physics/

Overview

Body objects are what interact with the physics world in Corona, are attached to DisplayObjects and are created using physics.addBody().
