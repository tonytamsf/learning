object:getAnchorB()

Type Function
Object Joint
Library physics.*
Return value Numbers
Revision Current Public Release (2013.2076)
Keywords getAnchorB
Sample code /CoronaSDK/SampleCode/
See also Joint:getAnchorA()

Overview

A function that returns the [x,y] coordinates of the Joint’s anchor points within objects A and B, which are the two joined objects. The values are given in the local coordinates of each object, so an anchor point at the center of the object would be at position [0, 0].

Syntax

Joint:getAnchorB()

Example

local x, y = myJoint:getAnchorA()
local x, y = myJoint:getAnchorB()
