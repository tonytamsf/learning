Overview

Paths specify the geometry of shapes. Path objects give you access to manipulate this geometry.

Properties

path.type

String. The type of path. See Types of Paths below.

Types of Paths

The type property can take on the following values:

  • "circle"
  • "rect"
  • "roundedRect"
  • ["polygon"][api.type.PolygonPath]
