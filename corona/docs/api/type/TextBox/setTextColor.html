object:setTextColor()

Type Function
Object TextBox
Library native.*
Return value none
Revision Current Public Release (2013.2076)
Keywords setTextColor, color, text

Overview

NOTE: This is deprecated. Please use object:setFillColor() instead.

Sets the color of the text in a text box.

Syntax

object:setTextColor( r, g, b )
object:setTextColor( r, g, b, a )
r, g, b (required)

Number. A value between 0 and 255 for red, green, and blue channels.

a (optional)

Number. A value between 0 and 255 for the alpha channel. Default is 255 (opaque).

Example

local textBox = native.newTextBox( 30, 140, 260, 150 )
textBox.font = native.newFont( "Helvetica-Bold", 18 )
textBox:setTextColor( 200, 250, 250, 255 )
textBox.alpha = 1.0
textBox.hasBackground = false
textBox.text = "If you've struggled with mobile development or are stumped where to begin, then you'll love Corona. Our APIs are simple and easy to use."
