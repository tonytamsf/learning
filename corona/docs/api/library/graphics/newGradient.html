graphics.newGradient()

Type Function
Library graphics.*
Return value Object (gradient)
Revision Current Public Release (2013.2076)
Keywords gradient, fill, color
See also object:setFillColor()
object:setTextColor().

Overview

NOTE: This is deprecated in graphics 2.0. Please look at the Shape Guide to learn how to use gradients with any shape.

Creates a gradient object that adds horizontal/vertical linear gradients to rectangle and text objects.

The gradient starts with color1 and goes to color2. Both color1 and color2 are table arrays.

This function returns a gradient object that is used with object:setFillColor() and object:setTextColor().

Gotchas

NOTE: This is only true of the old graphics 1.0 engine.

Gradients do not work with objects created with display.newCircle(), display.newRoundedRect(), or display.newGroup().

Syntax

graphics.newGradient( color1, color2 [ , direction] )
color1 (required)

Table. Table array with the starting gradient color. The table format for both color1 and color2 can be one of the following:

{ gray } { gray, alpha } { red, green, blue } { red, green, blue, alpha }

Gray, alpha, red, green, and blue are values in the range of 0 to 255.

color2 (required)

Table. Table array with the ending gradient color.

direction (optional)

String. String containing the gradient direction. Defaults to "down" if not supplied.

"down" gradient starts at the top and goes to the bottom (default) "up" gradient starts at bottom and goes to the top "left" gradient starts at the right and goes to the left "right" gradient starts at the left and goes to the right

Example

local g = graphics.newGradient(
  { 255, 255, 255 },
  { 200, 200, 200 },
  "down" )

-- sets gradient 'g' on rect
local rect = display.newRect( 0, 0, 100, 200 )
rect:setFillColor( g ) 

-- sets gradient 'g' on text
local text = display.newText("Hello World!", 0, 0, native.systemFont, 16)
text:setTextColor( g )
