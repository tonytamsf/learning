display.newLine()

Type Function
Library display.*
Return value LineObject
Revision Current Public Release (2013.2076)
Keywords lines, vector
Sample code /CoronaSDK/SampleCode/Graphics/Polylines
See also display.setDefault()
object.width
object:append()

Overview

Draw a line from one point to another. Optionally, append points to the end of the line.

Gotchas

  • Currently display.newLine() objects do not support touch events (case #3803).

  • By default, the fill color is white (rgb 255, 255, 255). The default color can be changed using display.setDefault().

Syntax

display.newLine( [parentGroup,] x1, y1, x2, y2 )
parentGroup (optional)

DisplayObject (group). Specify an optional display group in which to insert the line. Uses the current stage (as returned from display.getCurrentStage()) if not specified.

x1, y1 (required)

Numbers. Coordinates of the beginning of the line.

x2, y2 (required)

Numbers. Coordinates of the end of the line.

Example

local star = display.newLine( 200,90, 227,165 )
star:append( 305,165, 243,216, 265,290, 200,245, 135,290, 157,215, 95,165, 173,165, 200,90 )
star:setColor( 255, 102, 102, 255 )
star.width = 3
