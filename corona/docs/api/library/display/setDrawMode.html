display.setDrawMode()

Type Function
Library display.*
Return value none
Revision Current Public Release (2013.2076)
Keywords wireframe
See also

Overview

Sets the draw mode.

Syntax

display.setDrawMode( key, value )
key (required)

String. Specifies the type of draw mode. See 'Keys' below.

Keys

The key corresponds to the draw mode. The following mode keys are supported:

"wireframe"

This turns on wireframe mode. Pass true to value to turn this mode on and false to restore normal rendering.

Examples

-- Turns on wireframe
display.setDrawMode( "wireframe", true )
-- Turns off wireframe
display.setDrawMode( "wireframe", false )
