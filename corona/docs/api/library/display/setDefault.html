display.setDefault()

Type Function
Library display.*
Return value none
Revision Current Public Release (2013.2076)
Keywords defaults, set default, color, graphics, vector objects
See also display.newCircle()
display.newRect()
display.newLine()
display.newText()
object.anchorX
[object.anchorY][pi.type.DisplayObject.anchorY]

Overview

Set defaults for display objects, such as default colors for display objects. Colors default to white if not set.

Syntax

Any one of the following are valid:

display.setDefault( key, r, g, b, alpha )
display.setDefault( key, r, g, b )
display.setDefault( key, gray )
display.setDefault( key, gray, alpha )
display.setDefault( key, value )
key (required)

String. Specifies the

gray, r, g, b, alpha (optional)

Numbers. Color values between 0 and 1

Display Object Keys

These keys specify default values common to all display objects.

"anchorX"

The anchor alignment along the x-direction. This value is what is used for the value of the anchorX property in new objects. The default is 0.5 which centers along the x-direction.

"anchorY"

The anchor alignment along the y-direction. This value is what is used for the value of the anchorY property in new objects. The default is 0.5 which centers along the y-direction.

Color Keys

A color key specifies which default color value to set:

"background"

Corresponds to the default color for that the screen is cleared with. The default is initially black. (Available to subscribers starting in daily build 2011.505)

"fillColor"

Corresponds to the default color for vector objects (e.g. display.newRect()). The default is initially white.

"strokeColor"

Corresponds to the default color for vector objects (e.g. display.newRect()). The default is initially white.

"lineColor"

Corresponds to the default color for line objects (e.g. display.newLine()). The default is initially white.

"textColor"

Corresponds to the default color for text objects (e.g. display.newText()). The default is initially white.

Examples

-- set default fill color for vector objects to red
display.setDefault( "fillColor", 255, 0, 0 )

-- set default screen background color to blue
display.setDefault( "background", 0, 0, 255 )

Texture Keys

"magTextureFilter"

The default magnification sampling filter applied whenever an image is loaded by Corona. Once an image is loaded the first time, the same sampling filter will be applied for any subsequent loads of the same file. This is because textures are loaded once per file. Default value is "linear". Other values are: "nearest".

"minTextureFilter"

The default minification sampling filter applied whenever an image is loaded by Corona. Once an image is loaded the first time, the same sampling filter will be applied for any subsequent loads of the same file. This is because textures are loaded once per file. Default value is "linear". Other values are: "nearest".

"textureWrapX"

The wrap mode along the x-direction. Once an image is loaded the first time, the same wrap mode will be applied for any subsequent loads of the same file. This is because textures are loaded once per file. Default value is "clampToEdge". Other values are useful for patterns: "repeat", "mirroredRepeat".

See the important note below.

"textureWrapY"

The wrap mode along the y-direction. Once an image is loaded the first time, the same wrap mode will be applied for any subsequent loads of the same file. This is because textures are loaded once per file. Default value is "clampToEdge". Other values are useful for patterns: "repeat", "mirroredRepeat".

See the important note below.

Important note on texture wrap

"clampToEdge" is the only valid texture wrap for non-power-of-two textures.

To use "repeat", make sure the dimentions of your texture are a power of two. For example 16, 32, 64, 128, 256, etc.

Examples

display.setDefault( "textureWrapX", "repeat" )
display.setDefault( "textureWrapY", "repeat" )

local x,y = display.contentCenterX, display.contentCenterY
local o = display.newRect( x, y, display.contentWidth, display.contentHeight )
o.fill = { type="image", filename="fish.small.red.png" }
o.fill.scaleX = 0.1
o.fill.scaleY = 0.1
