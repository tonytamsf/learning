display.newImageGroup()

Type function
Library display.*
Return value GroupObject
Revision Current Public Release (2013.2076)
Keywords migration
See also display.newGroup()

Overview

This function has been removed. In the v1 graphics engine, this was used to optimize batching for objects that shared the same underlying GPU texture.

Migration

In the v2 graphics engine, you can achieve the same level of batching by placing the same objects as children in a group. As long as the objects all use the same image (or imagesheet), the engine will be able to batch them as before.

As with the old image groups, there are similar limitations:

  • Masks cannot be applied individually to objects that are inserted into an image group.
  • No per-object blend mode (for objects inserted into the image group).
  • Per-object effects that use too many parameters will break the batch.
