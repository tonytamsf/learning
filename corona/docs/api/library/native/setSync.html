native.setSync()

Type Function
Library native.*
Return value Boolean
Revision Current Public Release (2013.2076)
Keywords sync, icloud, backup, documents
See also

Overview

Sets the iCloud automatic backup flag for files in the system.DocumentsDirectory on Mac OS X and iOS systems.

The iCloud backup is true by default and must be set false with this API to disable automatic backup of files to iCloud.

This function returns up to two values:

  • value - Boolean: true if setSync was successful, false if setSync failed. Returns nil on Android and Windows systems since they don't support this API.
  • errStr - String: Error message if "value" returned false.

Gotchas

This is for Mac OS X and iOS platforms only.

Syntax

native.setSync( filename, params )
filename (required)

String. Name of file (including path) of file.

params (required)

Table. { iCloudBackup = flag }, The flag is true to enable backup to iCloud (default case) or false to disable backup.

Example

The following assumes a file named "SavedData.dat" is in the Documents directory:

-- Disable file backup
local results, errStr = native.setSync( "SavedData.dat", { iCloudBackup = false } ) 

-- Disable file backup in a subdirectory within the Documents directory
local results, errStr = native.setSync( "data/SavedData.dat", { iCloudBackup = false } )
