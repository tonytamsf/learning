gameNetwork.init()

Type Function
Library gameNetwork.*
Return value none
Revision Current Public Release (2013.2076)
Keywords gameNetwork, game center
Sample code /CoronaSDK/SampleCode/GameNetwork/GameCenter
See also gameNetwork.request()

Overview

Initializes an app for use with the game network provider.

NOTE: Using the gameNetwork library will enable Corona Analytics and override any setting in config.lua.

Syntax

gameNetwork.init( providerName [, ...] )
providerName (required)

String. Name of the gameNetwork provider to initialize.

  • "gamecenter": Only available on iOS.

Providers

Game Center

Game Center is a service provided by Apple and is only available on iOS.

gameNetwork.init( "gamecenter" [, initCallback] )
initCallback (optional)

Listener. If "gamecenter" is specified as providerName, this is a callback function. On successful login, event.data will be 'true'. On unsuccessful init, event.data will be 'false'. When problems such as network errors occur, event.errorCode (integer) and event.errorMessage (string) will be defined. Starting in iOS 6, there is an event.type called "showSignIn" which will be invoked before the Game Center login screen appears. This gives you a chance to pause your game or do any special actions you need before the login view takes over the screen. event.data will return the login status (presumably false) so existing code not checking for this type will still work. When this view is dismissed, the init callback will be invoked again with event.type set to "init" (same as pre-iOS 6) to handle the results of the login.

Also be aware that iOS backgrounding will cause your app to automatically log out the user from Game Center. When the app is resumed, Game Center will automatically try to re-login the user. The callback function you specified here will be invoked again providin the result of that re-login attempt. Thus, this callback function exists for the life of your application. With Game Center, it is advisable to avoid calling other Game Center functions when the user is not logged in.

Google Play Game Services

Full Google Play Game Services implementation is currently underway in Corona. For more information, please refer to the plugin documentation here.

Example

local gameNetwork = require "gameNetwork"

local function initCallback( event )
    -- "showSignIn" is only available on iOS 6+
    if event.type == "showSignIn" then
        -- This is an opportunity to pause your game or do other things you might need to do while the Game Center Sign-In controller is up.
        -- For the iOS 6.0 landscape orientation bug, this is an opportunity to remove native objects so they won't rotate.
        -- This is type "init" for all versions of Game Center.
    elseif event.data then
        loggedIntoGC = true
        native.showAlert( "Success!", "", { "OK" } )
    end
end

local function onSystemEvent( event ) 
    if event.type == "applicationStart" then
        gameNetwork.init( "gamecenter", initCallback )
        return true
    end
end
Runtime:addEventListener( "system", onSystemEvent )
