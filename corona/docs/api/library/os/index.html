os.*

Type Library
Revision Current Public Release (2013.2076)
Keywords os, operating system, time, date, clock
See also system.*

Overview

This standard Lua library provides functions for dealing with system time and date and other OS-related functions.

For more information, see the Working With Time and Dates tutorial.
