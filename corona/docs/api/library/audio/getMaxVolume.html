Overview

Gets the max volume for a specific channel. NOTE: There is no max volume for the master volume.

This function returns the channel max volume if requesting a channel volume, as well as the average max volume across all channels if channel=0 is specified.

Syntax

audio.getMaxVolume( { channel=c } )

This function takes a single table as a parameter, with the following optional keys:

channel (optional)

Number. The channel number you want to get the max volume on. 1 to the maximum number of channels (currently 32) are valid channels. Specifying 0 will return the average volume across all channels.

Example

Get the max volume on channel 1:

local channel1MaxVolume = audio.getMaxVolume( { channel=1 } )
