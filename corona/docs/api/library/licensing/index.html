Overview

The Corona licensing library lets you check to see if the app was bought from a store. Currently, only one store is supported:

  • Google Play

Functions

licensing.init()

licensing.verify()

Example

The following code demonstrates basic usage of the licensing library. This example targets the Google Play Store.

config.lua

Google

The key is a value obtained from Google.

The policy statement is optional. It can either be "serverManaged" (default value) or "strict". "serverManaged" will query the Google server and cache the results. It is similar to the ServerManagedPolicy on Google's reference page. "strict" will not cache the results, so when there is a network failure, the licensing will fail. It is similar to the StrictPolicy on Google's reference page.

application =
{
    content =
    {
        width = 320,
        height = 480,
        scale = "letterbox"
    },
    license =
    {
        google =
        {
            key = "Your key here",
            policy = "this is optional",
        },
    },
}

main.lua

local licensing = require "licensing"
licensing.init( "google" )

local function licensingListener( event )
    local verified = event.isVerified
    if not event.isVerified then
        -- Failed verify app from the play store, we print a message
        print("Pirates!!!")
    end
end

licensing.verify( licensingListener )
