math.floor()

Revision Current Public Release (2013.2076)

Sample code /CoronaSDK/SampleCode/

Overview

Returns the largest integer smaller than or equal to x.

Syntax

math.floor( x )
x (required)

Number. A number.

Examples

print(math.floor(0.5))    ---> 0
print(math.floor(-0.5))    ---> -1
