widget.newScrollView()

Type Function
Library widget.*
Return value ScrollViewWidget
Revision Current Public Release (2013.2076)
Keywords widget, scrollview, scrollbox, scrolling content
Sample code /CoronaSDK/SampleCode/Interface/WidgetDemo
See also ScrollViewWidget
graphics.newMask()

Overview

Creates a ScrollView object.

Limitations

The ScrollView object uses one mask to define it's shape. You need to consider this if you are adding this object to a container (which also uses a mask) or adding masked items to the object (i.e., text objects). Graphcis 2.0 has a mask nesting limit of three.

For a walkthrough on how to use image masks, see the Masking Images guide.

This widget does not support scaling.

Syntax

widget.newScrollView( options )

This function takes a single argument, options, which is a table that accepts the following parameters:

id (optional)

String. This is an optional id to assign to your ScrollView. Default is "widget_scrollView".

left, top (optional)

Numbers. This is the position where the ScrollView will be created. The default values for left and top are 0, 0.

width, height (required)

Numbers. This is the width/height of the ScrollView.

scrollWidth, scrollHeight (required)

Numbers. This is the width/height of the total scrollable content area. This cannot be changed after the ScrollView widget has been created.

topPadding, bottomPadding (optional)

Numbers. These numbers represent the number of pixels from the top and bottom of the ScrollView in which content will stop when it reaches the top/bottom of scrollable area. The default value for both is 0.

leftPadding, rightPadding (optional)

Numbers. These numbers represent the number of pixels from the left and right of the ScrollView in which content will stop when it reaches the left/right of scrollable area. The default value for both is 0.

friction (optional)

Number. This determines how fast the content travels when it is flicked up or down. The default value is 0.972.

backgroundColor (optional)

Table. This is a table that holds the color (in rbga format) of the ScrollView background. Default is white:

{ 255, 255, 255, 255 }
hideBackground (optional)

Boolean. If set to true, the background of the ScrollView will be not be visible.

horizontalScrollDisabled (optional)

Boolean. If set to true, the ScrollView will be unable to scroll horizontally.

verticalScrollDisabled (optional)

Boolean. If set to true, the ScrollView will be unable to scroll vertically.

isLocked (optional)

Boolean. If set to true, the ScrollView will be locked, meaning it cannot be scrolled vertically or horizontally.

hideScrollBar (optional)

Boolean. Set this to true if you do not want to use a scrollBar in your ScrollView.

listener (optional)

Listener. Used to listen for ScrollView events, with the following events: event.limitReached - Indicates that the scrollView has reached one of it's limits. event.direction - Returns the direction the scrollView is moving in.

Event phases of: began, moved & ended. In your listener, event.target is a reference to the ScrollView object that the listener is attached to. See the example for how to construct the listener function.

baseDir (optional)

Constant. This is the path to your maskFile (if using one). Default is system.ResourceDirectory. See system.pathForFile() for valid values.

maskFile (optional)

String. If you set a custom width/height, you need to provide a maskFile, which is the filename of the image mask which will be used on your ScrollView.

isBounceEnabled (optional)

Boolean. Set this to false if you do not want your ScrollView to bounce back when reaching a limit (top, bottom, left, right), but simply stop scrolling when it reaches that limit. Defaults to true.

Visual Customization

scrollBarOptions (optional)

Table. This is a table that specifies the imageSheet object for the scrollBar, and it's top, middle and bottom frames. If not specified the scrollBar will be created using the default widget theme.

Format for scrollBarOptions

Scrollbar's are constructed in the form of a 3 slice image. You must specify all of the parameters below if you wish to visually customize your scrollBar.

sheet (required)

ImageSheet. This is the image sheet object for your scrollBar.

topFrame (required)

Number. This is the top frame index of the scrollBar.

middleFrame (required)

Number. This is the middle frame index of the scrollBar.

bottomFrame (required)

Number. This is the bottom frame index of the scrollBar.

Example

local widget = require( "widget" )

-- Our ScrollView listener
local function scrollListener( event )
    local phase = event.phase
    local direction = event.direction

    if "began" == phase then
        print( "Began" )
    elseif "moved" == phase then
        print( "Moved" )
    elseif "ended" == phase then
        print( "Ended" )
    end

    -- If we have reached one of the scrollViews limits
    if event.limitReached then
        if "up" == direction then
            print( "Reached Top Limit" )
        elseif "down" == direction then
            print( "Reached Bottom Limit" )
        elseif "left" == direction then
            print( "Reached Left Limit" )
        elseif "right" == direction then
            print( "Reached Right Limit" )
        end
    end

    return true
end

-- Create a ScrollView
local scrollView = widget.newScrollView
{
    top = 100,
    left = 10,
    width = 300,
    height = 350,
    scrollWidth = 465,
    scrollHeight = 670,
    listener = scrollListener,
}

-- Create a image and insert it into the ScrollView
local background = display.newImageRect( "assets/scrollimage2.jpg", 768, 1024 )
background.x = bg.contentWidth * 0.5
background.y = bg.contentHeight * 0.5
scrollView:insert( background )
