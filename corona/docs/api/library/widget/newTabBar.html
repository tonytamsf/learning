widget.newTabBar()

Type Function
Library widget.*
Return value TabBarWidget
Revision Current Public Release (2013.2076)
Keywords widget, tab bar, tabs, tab buttons
See also TabBarWidget

Overview

Creates a TabBar object.

TabBar buttons will be auto-positioned inside the TabBar.

Gotchas

TabBars do not support scaling nor changing it's width/height via .width/.height.

TabBars also do not support changing reference points after creation.

Syntax

widget.newTabBar( options )

This function takes a single argument, options, which is a table that accepts the following parameters:

id (optional)

String. This is an optional id to assign to your TabBar. Default is "widget_tabBar".

left, top (optional)

Numbers. This is the position where you want your TabBar to be created. The default values for left and top are 0, 0.

width, height (optional)

Numbers. Set these if you want to use a custom width/height for your tab bar. The default width is equal to display.contentWidth, and the default height is 50.

buttons (required)

Table. This is a table that will hold all parameters and options for the individual tab buttons. Please see the Format for buttons section below for more information.

Visual Customization

Image File Tab Bar Creation

baseDir (optional)

Constant. This is the base directory where your custom images are located. The default is your project folder (system.ResourceDirectory).

backgroundFile (required)

String. This is the background image of the TabBar.

tabSelectedLeftFile (required)

String. This is the left edge image of the TabBar's selected graphic.

tabSelectedMiddleFile (required)

String. This is the middle image of the TabBar's selected graphic.

tabSelectedRightFile (required)

String. This is the right edge image of the TabBar's selected graphic.

tabSelectedFrameWidth, tabSelectedFrameHeight (required)

Numbers. This is the width/height of the TabBar's selected graphic.

ImageSheet Tab Bar Creation

sheet (required)

ImageSheet. This is the image sheet object for your TabBar.

backgroundFrame (required)

Number. This is the background frame index of the TabBar.

backgroundFrameWidth, backgroundFrameHeight (required)

Numbers. This is the width/height of the background image of the TabBar.

tabSelectedLeftFrame (required)

Number. This is the left edge frame index of the TabBar's selected graphic.

tabSelectedMiddleFrame (required)

Number. This is the left edge frame index of the TabBar's selected graphic.

tabSelectedRightFrame (required)

Number. This is the left edge frame index of the TabBar's selected graphic.

tabSelectedFrameWidth, tabSelectedFrameHeight (required)

Numbers. This is the width/height of the TabBar's selected graphic.

Format for buttons

This is a table that will hold all parameters and options for the individual tab buttons.

id (optional)

String. This is an optional id to assign to your TabBar button. Default is "button".

label (optional)

String. This is the text label that will (optionally) underneath the TabBar button icon.

labelColor (optional)

Table. This is a table that holds the color (in rbga format) of the TabBar button label. Example:

{
    default = { 255, 255, 255, 90 },
    over = { 120, 53, 128, 255 },
}
labelXOffset, labelYOffset (optional)

Numbers. This is a x/y offset you want to apply to the TabBar label.

font (optional)

String. This is the font to use for your TabBar button label. The default is native.systemFont

size (optional)

Number. The font size (in pixels) for your TabBar button label. The default is 8.

selected (optional)

Boolean. Set this to true to start the TabBar button in it's selected state. You should only set one of these to true at a time.

onPress (optional)

Function. This is an optional function to be called as soon as the button is pressed. This callback function does not require you to test for an event.phase (as it only has one) and should return true.

Button Creation From Image Files

width, height (required)

Numbers. This should match the width/height of your defaultFile/overFile.

baseDir (optional)

Constant. This is the base directory where your custom images are located. The default is your project folder (system.ResourceDirectory).

defaultFile (required)

String. This is the default ("un-pressed") image of the TabBar button.

overFile (required)

String. This is the over ("pressed") image of the TabBar button.

Button Creation From Image Sheet

defaultFrame (required)

Number. This is the default ("un-pressed") frame index of the TabBar button.

overFrame (required)

Number. This is the over ("pressed") frame index of the TabBar button.

Example

local widget = require( "widget" )

-- Create the tabBar's buttons
local tabButtons = 
{
    {
        width = 32, 
        height = 32,
        defaultFile = "assets/tabIcon.png",
        overFile = "assets/tabIcon-down.png",
        label = "Tab1",
        labelColor =
        {
            default = { 0, 0, 0 },
            over = { 255, 255, 255 },
        },
        font = native.systemFontBold,
        size = 8,
        onPress = function() print( "Tab 1 pressed" ) end,
        selected = false
    },
    {
        width = 32, 
        height = 32,
        defaultFile = "assets/tabIcon.png",
        overFile = "assets/tabIcon-down.png",
        label = "Tab2",
        labelColor =
        {
            default = { 0, 0, 0 },
            over = { 255, 255, 255 },
        },
        font = native.systemFontBold,
        size = 8,
        onPress = function( event ) print( "Tab 2 pressed" ); print( event.target._id ) end,
        selected = true
    },
    {
        width = 32, 
        height = 32,
        defaultFile = "assets/tabIcon.png",
        overFile = "assets/tabIcon-down.png",
        label = "Tab3",
        labelColor =
        {
            default = { 0, 0, 0 },
            over = { 255, 255, 255 },
        },
        font = native.systemFontBold,
        size = 8,
        onPress = function() print( "Tab 3 pressed" ) end,
        selected = false
    },
    {
        width = 32, 
        height = 32,
        defaultFile = "assets/tabIcon.png",
        overFile = "assets/tabIcon-down.png",
        label = "Tab4",
        labelColor =
        {
            default = { 0, 0, 0 },
            over = { 255, 255, 255 },
        },
        font = native.systemFontBold,
        size = 8,
        onPress = function() print( "Tab 4 pressed" ) end,
        selected = false
    },
}

-- Create a tabBar
local tabBar = widget.newTabBar
{
    left = 0,
    top = display.contentHeight - 52,
    width = display.contentWidth,
    backgroundFile = "assets/woodbg.png",
    tabSelectedLeftFile = "assets/tabBar_tabSelectedLeftEdge.png",
    tabSelectedRightFile = "assets/tabBar_tabSelectedRightEdge.png",
    tabSelectedMiddleFile = "assets/tabBar_tabSelectedMiddle.png",
    tabSelectedFrameWidth = 20,
    tabSelectedFrameHeight = 52,
    buttons = tabButtons,
}
group:insert( tabBar )
