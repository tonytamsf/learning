widget.newProgressView()

Type Function
Library widget.*
Return value ProgressViewWidget
Revision Current Public Release (2013.2076)
Keywords widget, progress, progressView, progressIndicator
See also ProgressViewWidget

Overview

Creates a ProgressView object.

Gotchas

To conserve texture memory, a Progress view can only be created from an imageSheet.

Progress views do not support Scaling nor changing it's width/height via .width/.height.

Progress views also do not support changing Reference Point's after creation.

Syntax

widget.newProgressView( options )

This function takes a single argument, options, which is a table that accepts the following parameters:

id (optional)

String. Can be set to anything you want, for easier identification of your widget. Default is "widget_progressView".

left, top (optional)

Numbers. This is the position where you want your Progress view to be created. The default values for left and top are 0, 0.

width (required)

Number. This is the overall width of your Progress view.

isAnimated (optional)

Boolean. Set this to true if the progress change should be animated, false (or omitted) if the progress change should happen immediately.

fillXOffset (optional)

Number. Set this if you wish to position your Progress view's fill image with a horizontal offset. Default is 0.

fillYOffset (optional)

Number. Set this if you wish to position your Progress view's fill image with a vertical offset. Default is 0.

Visual Customization

sheet (required)

ImageSheet. This is the image sheet object for your Progress view.

fillOuterLeftFrame (required)

Number. This is the frame index for the Progress view outer left frame (this frame is the progress view's container/background left edge segment).

fillOuterMiddleFrame (required)

Number. This is the frame index for the Progress view outer left frame (this frame is the progress view's container/background middle segment).

fillOuterRightFrame (required)

Number. This is the frame index for the Progress view outer left frame (this frame is the progress view's container/background right edge segment).

fillInnerLeftFrame (required)

Number. This is the frame index for the Progress view inner left fill frame (this frame is the left edge segment of the progress view's fill).

fillInnerMiddleFrame (required)

Number. This is the frame index for the Progress view inner middle fill frame (this frame is the middle edge segment of the progress view's fill).

fillInnerRightFrame (required)

Number. This is the frame index for the Progress view inner right fill frame (this frame is the right edge segment of the progress view's fill).

fillOuterWidth, fillOuterHeight (required)

Numbers. This is the width/height of the outer frame of your Progress view. All fill frames should be equal width/height.

fillWidth, fillHeight (required)

Numbers. This is the width/height of the fill frames of your Progress view. All fill frames should be equal width/height.

Example

local widget = require( "widget" )

-- Create a new progressView
local newProgressView = widget.newProgressView
{
    left = 150,
    top = 200,
    width = 400,
    isAnimated = true,
}
