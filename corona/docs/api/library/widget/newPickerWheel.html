widget.newPickerWheel()

Type Function
Library widget.*
Return value PickerWidget
Revision Current Public Release (2013.2076)
Keywords widget, picker, pickerWheel, wheel
See also PickerWidget

Creates a Picker Wheel object.

Gotchas

To conserve texture memory, a Picker Wheel can only be created from an imageSheet.

This widget does not support scaling.

Syntax

widget.newPickerWheel( options )

This function takes a single argument, options, which is a table that accepts the following parameters:

id (optional)

String. This is an optional id to assign to your Picker Wheel. Default is "widget_pickerWheel".

left, top (optional)

Numbers. This is the position where the Picker Wheel will be created. The default values for left and top are 0, 0.

font (optional)

String. This is the font that will be used when rendering column rows. Default is native.systemFontBold.

fontSize (optional)

Number. The font size (in pixels) for your Picker Wheel columns. The default is 22.

fontColor (optional)

Table. This is a table that holds the color (in rbga format) of your Picker Wheel columns. Example:

{ 0, 0, 0, 255 }
columnColor (optional)

Table. This is a table that holds the color (in rbga format) of your Picker Wheel columns. The default is white:

{ 255, 255, 255, 255 }
columns (optional)

Table. This is a table that should have sub-tables representing the individual columns in your picker wheel. The numerical keys in each column table will represent the individual rows (see Example section). Additionally, each column table can have the following properties:

  • width: Number. By default, columns are split evenly between the pickerWheel's viewable area, but there are some cases where you want some columns to be wider than the others. Use the width property to set a custom width on a per-column basis.
  • labels: Table. This is a table which contains a label for each row. Ex: { "One", "Two", "Three", "Four" }
  • startIndex: Number. If you want a column to start at a specific row, specify the index number of that row here.
  • align: String. This can either be set to right, left or center. If this property is not set, text will be center-aligned for that column by default.

Visual Customization

sheet (required)

ImageSheet. This is the image sheet object for your Picker Wheel.

backgroundFrame (required)

Number. This is the background frame index that will sit behind the Picker Wheel overlay (glass image). This image will be stretched to the full width/height of the Picker Wheel.

backgroundWidth, backgroundHeight (required)

Numbers. This is the width/height of the Picker Wheel background frame image.

overlayFrame (required)

Number. This is the overlay frame index (glass image) of the Picker Wheel.

overlayFrameWidth, overlayFrameHeight (required)

Numbers. This is the width/height of the Picker Wheel overlay frame image.

separatorFrame (required)

Number. This is the separator (divider) frame index that separates each Picker Wheel column.

separatorWidth, separatorHeight (required)

Numbers. This is the width/height of the Picker Wheel separator frame image.

maskFile (required)

String. This is the maskFile used to crop the Picker Wheel columns.

Example

local widget = require( "widget" )

-- Create two tables to hold our days & years      
local days = {}
local years = {}

-- Populate the days table
for i = 1, 31 do
    days[i] = i
end

-- Populate the years table
for i = 1, 44 do
    years[i] = 1969 + i
end

-- Set up the Picker Wheel's columns
local columnData = 
{ 
    { 
        align = "right",
        width = 150,
        startIndex = 5,
        labels = 
        {
            "January", "February", "March", "April", "May", "June", "July", "August", "September", "October", "November", "December" 
        },
    },

    {
        align = "left",
        width = 60,
        startIndex = 18,
        labels = days,
    },

    {
        align = "center",
        width = 80,
        startIndex = 10,
        labels = years,
    },
}

-- Create a new Picker Wheel
local pickerWheel = widget.newPickerWheel
{
    top = display.contentHeight - 222,
    font = native.systemFontBold,
    columns = columnData,
}
