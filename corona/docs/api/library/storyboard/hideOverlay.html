storyboard.hideOverlay()

Type Function
Library storyboard.*
Return value none
Revision Current Public Release (2013.2076)

Sample code /CoronaSDK/SampleCode/Interface/Storyboard

overlayBegan event
overlayEnded event

Overview

This function will hide/remove the current overlay scene (if one is currently being displayed). Overlay (e.g. "pop up") scenes can be shown using storyboard.showOverlay().

Syntax

storyboard.hideOverlay( [ purgeOnly, effect, effectTime ] )

-- also supported:

storyboard.hideOverlay( [ effect, effectTime ] )
purgeOnly (optional)

Boolean. If set to true, the scene will only be purged (e.g. the view display group will be removed, but the scene module will remain in memory). Default is false (overlay scene will be completely removed, module and all). Setting this to true is useful if you intend to show the popup again at some point in the near future.

effect (optional)

String. If you want the overlay to have an effect before being removed, specify a scene effect here. All scene effects from storyboard.gotoScene() are supported.

effectTime (optional)

Number. The duration of time the effect should last. This is only taken to account if the effect is also set. Default: 500 (milliseconds).

Example

-- hide the currently shown overlay (if there is one)
storyboard.hideOverlay( "fade", 800 )

-- or ...

-- purgeOnly will purge the scene instead of completely removing it
storyboard.hideOverlay( true, "fade" )

-- or ...
