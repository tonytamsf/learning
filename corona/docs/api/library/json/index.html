Overview

The json library allows you serialize and deserialize Lua tables into JSON (JavaScript Object Notation) and vice-versa. You typically use JSON-formatted data in conjunction with web APIs, e.g. requests sent in the facebook library.

Usage

The JSON library follows new conventions when using require. You must store the result in a local variable:

local json = require "json"
json.encode( { key=value } )    -- OKAY

Do not assume that require will automatically assign a global:

require "json"
json.encode( { key=value } )    -- ERROR: global 'json' is nil
