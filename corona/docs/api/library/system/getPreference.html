system.getPreference()

Type Function
Library system.*
Return value String
Revision Current Public Release (2013.2076)
Keywords system preference

Overview

Returns a preference value as a string.

Syntax

system.getPreference( category, name )
category (required)

String. Valid values of category are: "ui" and "locale".

name (required)

String. Valid values depend on category.

For the "ui" category: "language".
For the "locale" category: "country", "identifier", and "language".

Gotchas

On iOS, "ui" language is set by the Language field on the General->International screen in the Settings app. The "locale" parameters are set by the Region Format field on the same screen.

Examples

print( system.getPreference( "locale", "country" ) )  -- print the locale country (eg. US)
print( system.getPreference( "locale", "identifier" ) )  -- print the locale language identifier (eg. us_US)
print( system.getPreference( "locale", "language" ) )  -- print the locale language (eg. en)
print( system.getPreference( "ui", "language" ) )  -- print the ui (device) language (eg. en)
