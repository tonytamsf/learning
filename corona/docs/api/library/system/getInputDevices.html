system.getInputDevices()

Type Function
Library system.*
Return value array of InputDevice
Revision Current Public Release (2013.2076)
Keywords InputDevice

Overview

Fetches all input devices that are currenty connected to the system, such as a touchscreen, keyboard, mouse, joystick, etc. Input devices that were once connected but later disconnected will not be returned by this function.

This function is intended to be called on application startup and on application resume. During runtime, it is best to use the inputDeviceStatus event instead which gets raised when new input devices have been connected to the system, when existing devices have been disconnected, and when devices have been reconnected.

Syntax

system.getInputDevices()

Return Value

Returns an array of InputDevices that are currently "connected" to the system.

Returns an empty array if no InputDevices were found.

Gotchas

Input device information is only provided on Android.

iOS, Mac, and Windows do not currently support input devices. This function will always return an empty array on these platforms.

Examples

-- Fetch all input devices currently connected to the system.
local inputDevices = system.getInputDevices()

-- Print all of the input devices found to the log.
if #inputDevices > 0 then
    print("Input Devices Found:")
    for index = 1, #inputDevices do
        print("- " .. inputDevices[index].displayName)
    end
else
    print("No input devices found.")
end
