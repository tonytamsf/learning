See also timer.resume()
timer.performWithDelay()

Overview

Pauses a timer started with timer.performWithDelay(). It returns a Number that represents the amount of time remaining in the timer.

Syntax

timer.pause( timerId )
timerId (required)

Object. The Timer ID from timer.performWithDelay().

Example

local function listener( event )
    print( "listener called" )
end

timer1 = timer.performWithDelay( 2000, listener )  -- wait 2 seconds
...
result = timer.pause( timer1 )
print( "Time remaining is " .. result )
