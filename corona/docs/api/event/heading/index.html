Type Event
Dispatch chain ▸ Runtime
Revision Current Public Release (2013.2076)
Keywords heading, compass
Sample code /CoronaSDK/SampleCode/
See also

Overview

These are events generated by the compass hardware, if available on the device.

Gotchas

Android

Android OS only supports event.magnetic, not event.geographic.

To enable heading events on Android devices, you must set the following permission in the "build.settings" file.

settings =
{
    android =
    {
        usesPermissions =
        {
            "android.permission.ACCESS_FINE_LOCATION"
        },
    },
}
