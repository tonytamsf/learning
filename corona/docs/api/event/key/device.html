event.device

Type InputDevice
Event key
Revision Current Public Release (2013.2076)
Keywords key, device

Overview

Provides access to the device that the key event came from.

Intended to be used by multiplayer games where multiple input devices will provide the same keys/buttons. This device property can be used to differentiate key events coming from different devices or players.

Gotchas

Will always return nil on iOS, Mac, and Windows. Information about external input devices is currently not supported on those platforms.

On Android, this property will return nil if the key event came from software, such as the virtual keyboard or the bottom virtual navigation bar.
