tap

Type Event
Dispatch chain ▸ Display Object ▸ Runtime
Revision Current Public Release (2013.2076)
Keywords tap
See also Tap/Touch/Multitouch
system.setTapDelay()

Overview

Generates a hit event when the user touches the screen. The event is dispatched to display objects in the display hierarchy. This is similar to the touch event except a hit count (number of taps) is available in the event callback.

Properties

event.name

event.numTaps

event.x

event.y

Examples

Function Listener
local object = display.newImage( "ball.png" )
object.id = "ball object"

local function onObjectTap( event )
    print( "Tap event on: " .. event.target.id )
    return true
end
object:addEventListener( "tap", onObjectTap )
