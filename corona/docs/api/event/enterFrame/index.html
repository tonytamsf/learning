Overview

enterFrame events occur at the frame interval of the application (30 or 60, as specified in your config.lua file). They are only dispatched to the global Runtime object.

Properties

event.name

event.time

Examples

event.name

local myListener = function( event )
    print( "Listener called with event of type " .. event.name )
end
Runtime:addEventListener( "enterFrame", myListener )

event.time

function printTimeSinceStart( event )
    print (event.time/1000 .. " seconds since app started." )
end 
Runtime:addEventListener("enterFrame", printTimeSinceStart)
