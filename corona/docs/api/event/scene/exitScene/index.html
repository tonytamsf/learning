Overview

This event is dispatched on the previously active scene just prior to the transition taking place. If an effect was specified for the scene transition, the exitScene event will be dispatched before the effect begins execution. If no effect is specified, the event will be the first to be dispatched after a call to storyboard.gotoScene().

This event is also dispatched on overlay scenes when [storyboard.hideOverlay()][api.library.storyboard.hideOverlay()] is called.
