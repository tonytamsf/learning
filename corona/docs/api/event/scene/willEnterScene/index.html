willEnterScene

Type Event
Dispatch chain ▸ Scene
Revision Current Public Release (2013.2076)

Sample code /CoronaSDK/SampleCode/Interface/Storyboard

storyboard.gotoScene()
storyboard.showOverlay()

Overview

This event is dispatched for the newly active Scene, just prior to a scene transition (initiated via storyboard.gotoScene() or storyboard.showOverlay()).

If a scene effect was specified for the scene transition, the willEnterScene event is dispatched before the scene effect begins execution. By contrast, the enterScene event is not dispatched until the effect is finished executing.
