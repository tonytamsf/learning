Overview

The sprite event is used by SpriteObjects and by the deprecated [SpriteInstance][api.type.SpriteInstance]

Sprite events are generated when a sprite instance animation has an event, such as when the animation stops, loops, or plays the next frame.

Gotchas

If you are porting from SpriteInstance to SpriteObject, the property event.sprite has changed to event.target.
