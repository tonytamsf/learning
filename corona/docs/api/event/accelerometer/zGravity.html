event.zGravity

Type Number
Event accelerometer
Revision Current Public Release (2013.2076)
Keywords accelerometer, zGravity

Overview

Provides the acceleration due to gravity relative to the z-axis of the device's screen in portrait orientation.

This value is "smoothed" based on the previous accelerometer readings in order to reduce a jittering effect when used to manipulate graphics on screen.

Gotchas

This value is always relative to the device in portrait orientation, regardless of the current orientation of your application. So, if your application is running in landscape mode, then you'll need to compensate by 90 degrees.
